StartGame

Start a new game.

Game.StartGame(system_path, start_time)

Parameters

system_pathA SystemBody to start at.  If this is a starport, the player will begin docked here; otherwise the player will begin in orbit around the specified body.
start_timeoptional, default 0.  Time to start at in seconds from the Pioneer epoch (i.e. from 3200-01-01 00:00 UTC).
