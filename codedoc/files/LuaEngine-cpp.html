Attributes

rand

The global Rand object.  Its stream of values will be different across multiple Pioneer runs.  Use this when you just need a random number and don’t care about the seed.

Availability

alpha 10

Status

stable

ticks

Number of milliseconds since Pioneer was started.  This should be used for debugging purposes only (eg timing) and should never be used for game logic of any kind.
