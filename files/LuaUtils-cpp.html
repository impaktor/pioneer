hash_random

static int l_hash_random(lua_State *L)
r = util.hash_random(seed, m, n)

Pick a number deterministically according to some input value.

accent = util.hash_random(person_seed .. 'accent', #accents)

Parameters

seedA string or a number.  The output is deterministic based on this value.
m, noptional.  If called as hash_random(seed), the result is in the range 0 <= x < 1.  If called as hash_random(seed, m, n), the result is an integer in the range m <= x <= n.
m must be less than n.  (nm) must be less than 2^32.

Availability

alpha 24
